CSS Sprite Navigation Example

This example demonstrates a two-column layout with:

  • Left Column: Navigation buttons using CSS sprites for hover effects
  • Right Column: Preview of the sprite image showing all button states

How it works:

  • The navigation buttons use different sections of a single sprite image
  • Each button has a normal state (top row) and hover state (bottom row)
  • The background position changes on hover to show different parts of the sprite
  • Using sprites reduces HTTP requests and improves page load time

Try hovering over the navigation buttons to see the sprite animation in action.

How Sprite Masking Works:

  • Each button (like the home button) is created by:
    • Setting a fixed width (46px) to show only one icon
    • Using background-position (0 0) to select which part of sprite to show
    • The rest of the sprite sheet is hidden outside the button's width
  • Example CSS for home button:
    • #home {
  width: 46px;  /* Only shows 46px width of the sprite */
  background: url('./img_navsprites_hover.gif') 0 0;  /* Starts at left edge (0 0) */
}
